# Shape Splats Particle Pack Vol. 1

## What This Pack Is

Shape Splats Particle Pack Vol. 1 is a texture pack for building stylized particles, impacts, splashes, decals, and UI/VFX accents. It includes original Luminary Labs splat textures in multiple sizes, animated flipbook variants, starter materials, and a demo scene for quick preview and setup.

## What's Included

- Static splat textures:
  - `ShapeSplats_256/`
  - `ShapeSplats_512/`
  - `ShapeSplats_1024/`
  - `ShapeSplats_2048/`
- Flipbook textures:
  - `ShapeSplats_Flipbooks/`
- Starter materials:
  - `ShapeSplats_Materials/`
- Demo content:
  - `ShapeSplats_Demo/ShapeSplats_Demo.unity`
  - `ShapeSplats_Demo/ShapeSplats_DemoPrefabs/`

`ShapeSplats_Media/` is internal publishing media for Asset Store submission and is not part of the normal customer usage workflow.

## How To Use It

1. Open `ShapeSplats_Demo/ShapeSplats_Demo.unity` to preview the pack.
2. Choose a texture resolution tier based on your target quality and performance needs.
3. Choose or duplicate a starter material from `ShapeSplats_Materials/`.
4. Assign a splat texture to your particle system, material, or effect setup.
5. If using flipbooks, enable **Texture Sheet Animation** and match the frame layout to the selected flipbook texture.
